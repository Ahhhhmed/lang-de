# 🇩🇪 German Learning Workspace Instructions

This file defines the naming and structure guidelines for files in this workspace.

---

## 📂 Project Structure & Naming

1. **Vocabulary:** Store files inside level subfolders (e.g., `vocabulary/A1/hobbies.md`).
2. **Grammar Notes:** Store files in the root folder with a level prefix (e.g., `grammar_notes/A1_verbs.md`).
3. **Exercises:** Store files in the root folder with a level prefix (e.g., `exercises/A1_verb_practice.md`).

---

## 📝 Vocabulary File Schema

All vocabulary files must divide words by type and use the following markdown table formats:

### 1. Nouns (Substantive)
Must include the correct definite article (`der`/`die`/`das`), plural form, and English translation:

| German (with article) | Plural Form | English Translation | Example Sentence |
| :--- | :--- | :--- | :--- |
| der Hund | -e | dog | *Der Hund ist lieb.* |

### 2. Verbs (Verben)
Must include key principal parts (3rd Person Present / Simple Past / Past Participle):

| German (Infinitive) | Principal Parts (3rd Sing / Past / Participle) | English | Example Sentence |
| :--- | :--- | :--- | :--- |
| sprechen | spricht / sprach / gesprochen | to speak | *Er spricht Deutsch.* |

### 3. Adjectives & Adverbs
| German | English | Example Sentence |
| :--- | :--- | :--- |
| schnell | fast | *Das Auto ist schnell.* |

### 4. Expressions & Phrases
| German Phrase | English Translation | Context / Usage |
| :--- | :--- | :--- |
| Wie geht's? | How's it going? | Informal greeting |
