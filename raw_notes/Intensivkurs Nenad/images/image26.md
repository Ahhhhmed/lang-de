### 2. Akkusativ Pronouns

| Nominativ | Akkusativ |
| :--- | :--- |
| ich | mich |
| du | dich |
| er | ihn |
| sie | sie |
| es | es |
| wir | uns |
| ihr | euch |
| sie / Sie | sie / Sie |

Examples:
*   Ich mag **dich**.
*   Du fragst **ihn**.
*   Ihr trefft **uns**.
*   Wir laden **euch** ein.
*   Wir sehen **sie**.

### 3. Verbs with Dativ

| Verb | Example |
| :--- | :--- |
| **glauben** | Ich glaube <span style="color: green; text-decoration: underline;">meinem Chef</span>. |
| **schmecken** | Das Essen schmeckt <span style="color: green; text-decoration: underline;">dem Gast</span>. |

### 4. Dativ Pronouns & Articles

| Nominativ | Dativ |
| :--- | :--- |
| ich | mir |
| du | dir |
| er | ihm |
| sie | ihr |
| es | ihm |
| wir | uns |
| ihr | euch |
| sie / Sie | ihnen / Ihnen |

| Gender / Number | Dativ Article |
| :--- | :--- |
| maskulin | dem |
| feminin | der |
| neutrum | dem |
| plural | den |

Example:
*   Das Essen schmeckt **mir / dir / ihm / ihr / uns / euch / ihnen**.
