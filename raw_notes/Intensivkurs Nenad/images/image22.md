# ❓ Indirekte Fragen (Indirect Questions)

> [Note: Board transforms direct questions into polite indirect questions using *ob* and W-question words.]

---

### 1. Direct Questions

| Type | Example |
| :--- | :--- |
| **W-Frage** | Warum ist ein Haar in meinem Essen? |
| **Ja-/Nein-Frage** | Kann ich einen neuen Teller haben? |

---

### 2. Indirect Question Patterns

| Direct Question Type | Connector | Example |
| :--- | :--- | :--- |
| W-Frage | same W-word | Darf ich fragen, <span style="color: red; text-decoration: underline;">warum</span> ein Haar in meinem Essen <span style="color: red; font-weight: bold;">ist</span>? |
| Ja-/Nein-Frage | <span style="color: red; text-decoration: underline;">ob</span> | Können Sie mir sagen, <span style="color: red; text-decoration: underline;">ob</span> ich einen neuen Teller haben <span style="color: red; font-weight: bold;">kann</span>? |

> In indirect questions, the conjugated verb goes to the end.

---

### 💬 Restaurant Complaint Examples

*   Darf ich fragen, **warum** ein Haar in meinem Essen <span style="color: red; font-weight: bold;">ist</span>?
*   Können Sie mir sagen, **ob** ich einen neuen Teller haben <span style="color: red; font-weight: bold;">kann</span>?
*   Ich möchte wissen, **wer** diesen Teller gemacht <span style="color: red; font-weight: bold;">hat</span>.
*   Ich möchte wissen, **wo** die Toilette <span style="color: red; font-weight: bold;">ist</span>.
*   Ich möchte wissen, **wo** ich reklamieren <span style="color: red; font-weight: bold;">kann</span>.
*   Ich möchte wissen, **was** das in meinem Teller <span style="color: red; font-weight: bold;">ist</span>.
*   Ich möchte wissen, **wann** ich einen neuen Teller <span style="color: red; font-weight: bold;">bekomme</span>.
*   Ich möchte wissen, **wie viel Uhr** es <span style="color: red; font-weight: bold;">ist</span>.
*   Ich möchte wissen, **wohin** ich für die Reklamation gehen <span style="color: red; font-weight: bold;">kann</span>.
*   Ich möchte wissen, **woher** das Haar <span style="color: red; font-weight: bold;">kommt</span>.

---

### 📌 Case Note

*   **Wo?** can combine with location prepositions and often uses Dativ: *in + Dativ* for static location.
