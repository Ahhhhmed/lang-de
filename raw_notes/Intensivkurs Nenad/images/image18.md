# 🎁 Verbs with Dativ + Akkusativ Objects

> [Note: Board lists verbs that can take two objects: a Dativ person and an Akkusativ thing.]

---

### 📊 Basic Rule

Some verbs answer two questions:

*   **Wem?** = person receiving/affected by the action → <span style="color: green; font-weight: bold;">Dativ</span>
*   **Was? / Wen?** = thing/person directly acted on → <span style="color: red; font-weight: bold;">Akkusativ</span>

Example:

| Sentence | Dativ Object | Akkusativ Object |
| :--- | :--- | :--- |
| Du gibst <span style="color: green; text-decoration: underline;">dem Mann</span> <span style="color: red; text-decoration: underline;">einen Apfel</span>. | dem Mann / ihm | einen Apfel / ihn |

---

| Verb | Example |

| **geben** | Gestern hast du <span style="color: green; text-decoration: underline;">mir</span> <span style="color: red; text-decoration: underline;">einen Apfel</span> gegeben. |
| **zeigen** | Am Wochenende hat sie <span style="color: green; text-decoration: underline;">mir</span> <span style="color: red; text-decoration: underline;">einen Trick</span> gezeigt. |
| **empfehlen** | Wir haben <span style="color: green; text-decoration: underline;">dir</span> <span style="color: red; text-decoration: underline;">ein Restaurant</span> empfohlen. |
| **bieten** | Ihr habt <span style="color: green; text-decoration: underline;">uns</span> <span style="color: red; text-decoration: underline;">einen Witz</span> geboten. |

### 🔁 Pronoun Replacement

| Full Object | Pronoun |
| :--- | :--- |
| <span style="color: green; text-decoration: underline;">dem Mann</span> | <span style="color: green; text-decoration: underline;">ihm</span> |
| <span style="color: red; text-decoration: underline;">einen Apfel</span> | <span style="color: red; text-decoration: underline;">ihn</span> |

Examples:
*   Du gibst <span style="color: green; text-decoration: underline;">dem Mann</span> <span style="color: red; text-decoration: underline;">einen Apfel</span>.
*   Du gibst <span style="color: green; text-decoration: underline;">ihm</span> <span style="color: red; text-decoration: underline;">einen Apfel</span>.
*   Du gibst <span style="color: red; text-decoration: underline;">ihn</span> <span style="color: green; text-decoration: underline;">dem Mann</span>.
