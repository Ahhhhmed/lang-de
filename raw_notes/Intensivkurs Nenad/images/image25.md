# 📍 Lokal vs. Direktional: *Wo?*, *Wohin?*, *Woher?*

> [Note: Board contrasts static location with directional movement and lists verbs for each category.]

---

### 📊 Core Difference

| Question | Meaning | Direction | Typical Verb Type |
| :--- | :--- | :---: | :--- |
| <span style="color: red; text-decoration: underline;">Wo?</span> | where? | static / local | sein, arbeiten, wohnen, bleiben |
| <span style="color: red; text-decoration: underline;">Wohin?</span> | where to? | movement toward | gehen, fahren, fliegen, reisen |
| <span style="color: red; text-decoration: underline;">Woher?</span> | where from? | movement from | kommen |

---

### 🧭 Local Verbs (*Wo?*)

*   **sein**
*   **arbeiten**
*   **wohnen**
*   **bleiben**

---

### ➡️ Directional Verbs (*Wohin? / Woher?*)

*   **kommen**
*   **fahren**
*   **gehen**
*   **fliegen**
*   **segeln**
*   **joggen**
*   **reisen**

> [Note: The board uses arrows to show movement toward and away from a place.]
