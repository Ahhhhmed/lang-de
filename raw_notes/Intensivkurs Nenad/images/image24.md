### 🔁 Reflexive Pronouns

| Pronoun | Akkusativ Object | Dativ Object |
| :--- | :--- | :--- |
| ich | <span style="color: red; text-decoration: underline;">mich</span> | <span style="color: red; text-decoration: underline;">mir</span> |
| du | <span style="color: red; text-decoration: underline;">dich</span> | <span style="color: red; text-decoration: underline;">dir</span> |
| er / sie / es | sich | sich |
| wir | uns | uns |
| ihr | euch | euch |
| sie / Sie | sich | sich |

> [Note: The board highlights that `ich/du` differ in Akkusativ vs. Dativ: **mich/mir**, **dich/dir**.]

### 🔗 Connectors and Word Order

| Connector | Type | English | Word Order |
| :--- | :--- | :--- | :--- |
| **dann** | Hauptsatz connector | then / after | Verb in position 2 |
| **denn** | Position 0 conjunction | because / since | normal subject-verb order |
| **dass** | Nebensatz connector | that | verb at the end |
| **weil** | Nebensatz connector | because | verb at the end |
| **wenn** | Nebensatz connector | when / if | verb at the end |

Board summary:
*   **HS:** `V2`
*   **NS:** `V am Ende`
