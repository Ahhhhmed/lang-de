# Sequencing Connectors (Zeitliche Reihenfolge)

Used to describe the **order of actions** or events.

## Connectors

| German | Meaning | Notes |
|--------|---------|-------|
| **Zuerst** | First of all | |
| **Dann** | Then | triggers inversion (P1) |
| **Danach** | After that | triggers inversion (P1) |
| **Anschliessend** | Subsequently | |
| **Im Anschluss** | Following that | |
| **Am Ende** | At the end (of an action) | |
| **Schlussendlich** | Finally / Eventually | |
| **Schliesslich** | Finally | also used in presentations |

---

## Example: Morning Routine

- **Zuerst** wachst du auf.
- **Dann** frühstückst du.
- **Danach** räumst du den Tisch ab.
- **Anschliessend** wäschst du ab.
- **Im Anschluss** trocknest du ab.
- **Dann** duschst du.
- **Danach** ziehst du dich an.
- **Am Ende** (vom Tag) ziehst du dich aus.
- **Schlussendlich** schläfst du ein.

---

## Word Order Note

When `dann` / `danach` appear at the beginning, the verb comes next (inversion):

```
danach  räumst  du...      ✓
danach  du räumst...       ✗

...und (du) räumst...      ✓  (with "und", no inversion)
```
