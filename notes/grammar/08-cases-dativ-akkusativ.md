# Cases: Dativ & Akkusativ — Verbs, Articles, Pronouns

## Verbs that take **Akkusativ** (direct object)

| Verb | Example |
|------|---------|
| kaufen | Ich kaufe **einen** Apfel. |
| machen | Du machst **einen** Kuchen. |
| mögen | Sie mag **den** Park. |
| haben | Wir haben **einen** Plan! |
| brauchen | Ihr braucht **ein** Croissant. |
| sehen | Sie haben **einen** Berg gesehen. |
| treffen | Ich habe meinen Freund getroffen. |
| einladen | Gestern hast du **mich** eingeladen. |
| bekommen | Gestern hat er ein Geschenk bekommen. |
| mieten | Wir haben ein Velo gemietet. |
| beantworten | Ihr habt die Frage beantwortet. |
| geniessen | Meine Eltern geniessen **mich**. |
| fragen | Er fragt den Mann nach^Dat der Zeit. |

---

## Verbs that take **Dativ** (indirect object)

| Verb | Example |
|------|---------|
| glauben | Ich glaube **meinem** Chef. |
| schmecken | Das Essen schmeckt **dem** Gast. |
| gefallen | Der Film hat **mir** gefallen. |
| gehören | Das Land hat **den** Leuten gehört. |
| passen | Der Termin hat **mir** nicht gepasst. |
| danken | Du hast **mir** gedankt. |
| helfen | Er hat **dir** geholfen. |
| antworten | Wir haben **unseren** Eltern geantwortet. |
| wehtun | Ihr habt **dem** Kind wehgetan. / Die Hand tut **mir** weh. |

---

## Verbs that take both **Dativ + Akkusativ**

| Verb | Example |
|------|---------|
| schenken | Ich schenke **meinem Freund**^Dat **einen Wein**^Akk. |
| geben | Er gibt **dem Kind**^Dat **ein Buch**^Akk. |
| bringen | Kannst du **mir**^Dat **das Brot**^Akk bringen? |
| zeigen | Ich zeige **dir**^Dat **das Foto**^Akk. |

> **Word order:** Dativ (indirect) comes **before** Akkusativ (direct)  
> unless the Akkusativ is a pronoun → pronoun comes first.

## Indirect Questions (Indirekte Fragen)

Use with `Darf ich fragen, ...` / `Können Sie mir sagen, ...` / `Ich möchte wissen, ...`

| Type | Connector | Example |
|------|-----------|---------|
| W-Frage | `warum`, `wer`, `was`, `wann`, `wo`, `wohin`, `woher`, `wie viel` | Darf ich fragen, **warum** ein Haar in meinem Essen ist? |
| Ja/Nein | `ob` | Können Sie mir sagen, **ob** ich einen neuen Teller haben kann? |

> **Word order in indirect questions:** Verb goes to the **end** of the subordinate clause.
