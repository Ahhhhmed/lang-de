# Prepositions (Präpositionen)

## Wechselpräpositionen (Two-way Prepositions)

These prepositions take **Dativ** for location (Wo?) and **Akkusativ** for direction (Wohin?).

| Preposition | + DAT (Wo? — location) | + AKK (Wohin? — direction) |
|-------------|------------------------|---------------------------|
| **in** | Das Buch ist im Regal. | Ich gehe **in** die Küche. |
| **an** | Er sitzt am Tisch. | Er geht **an** den Tisch. |
| **auf** | Die Katze ist auf dem Sofa. | Die Katze springt **auf** das Sofa. |
| **neben** | Er steht neben dem Auto. | Er stellt das Rad **neben** das Auto. |
| **über** | Das Bild hängt über dem Bett. | Er hängt das Bild **über** das Bett. |
| **unter** | Der Hund liegt unter dem Tisch. | Der Hund läuft **unter** den Tisch. |
| **vor** | Er wartet vor dem Eingang. | Er geht **vor** den Eingang. |
| **hinter** | Das Auto ist hinter dem Haus. | Das Auto fährt **hinter** das Haus. |
| **zwischen** | Er sitzt zwischen den Stühlen. | Er setzt sich **zwischen** die Stühle. |

## Wo? vs. Wohin? / Woher?

| Wo? (location) | Wohin? → / Woher? ← (direction) |
|----------------|----------------------------------|
| lokal — static | direktional — movement |
| sein, arbeiten, wohnen, bleiben | kommen, fahren, gehen, fliegen, segeln, joggen, reisen |

### 1. Wohin gehst du? (Direction / Movement → Akkusativ, except *zu*)

| Preposition | Target / Category | Example Sentence |
|-------------|-------------------|------------------|
| **in** (+ Akk) | Buildings, rooms, countries with article | Ich gehe **ins** (in + das) Büro. / **ins** Krankenhaus. / **ins** Kino. / **ins** Bett. / **in die** Bar. / **in die** Schweiz. / **in die** Kita. / **in die** Apotheke. / **in den** Coop. / **in die** Migros. / **in die** Schule. |
| **zu** (+ Dat) | Persons, institutions | Ich gehe **zu** Freunden. / **zur** (zu + der) Arbeit. / **zum** (zu + dem) See. / **zum** Bahnhof. / **zu den** Eltern. / **zur** Kita. / **zur** Station. / **zu einem** Konzert. |
| **nach** | Cities, countries without article, home | Ich gehe **nach** Zürich. / **nach** Mexiko. / **nach** Hause. |
| **an** (+ Akk) | Waterways, events, specific meeting spots | Ich gehe **an den** Match. / **an den** See. / **an die** Bar. |

> [!TIP]
> - **in + Akkusativ** = Buildings, rooms, countries (with article).
> - **zu + Dativ** = People, institutions.
> - **nach** = Cities, countries (without article), "nach Hause".
> - **an + Akkusativ** = Waterways, events, certain meeting points.

---

### 2. Wo bist du? (Location / Static → Dativ)

| Preposition | Target / Category | Example Sentence |
|-------------|-------------------|------------------|
| **in** (+ Dat) | Buildings, countries, cities, rooms | Ich bin **im** (in + dem) Büro. / **im** Tram. / **im** Krankenhaus. / **im** Kino. / **im** Bett. / **im** Bus. / **in der** Migros. / **in der** Apotheke. / **in der** Bar. / **in der** Schweiz. / **in** Zürich. / **in der** Kita. |
| **bei** (+ Dat) | People, work, home | Ich bin **bei der** Arbeit. / **bei** Freunden. / **beim** (bei + dem) Arzt. / **bei den** Eltern. / **zu** Hause (note: static home is *zu Hause*). |
| **an** (+ Dat) | Waterways, points/stations, events | Ich bin **an der** Bar. / **am** (an + dem) See. / **an der** Station / **an der** Haltestelle. / **an einem** Konzert. |

> [!TIP]
> - **in + Dativ** → Buildings, countries, cities, rooms.
> - **bei + Dativ** → Persons, work, home (*zu Hause*).
> - **an + Dativ** → Waterways, meeting points, stations.

## „hin" und „her"

| | Meaning | Question |
|--|---------|----------|
| **hin** | away from speaker, towards destination | **Wo**hin? |
| **her** | towards speaker, away from origin | **Wo**her? |

**With verbs:**
```
gehen  + hin  →  hingehen
kommen + her  →  herkommen
```

---

## Sentence Order: TOP (Zeit — Objekt — Ort)

Time → Object → Place

- **Im Winter** fahre ich gern **Ski** **in den Bergen**.
- Fährst du gern **im Winter** Ski **in den Bergen**?
- **Im Frühling** lese ich immer **in Zürich am See**.
- **Im Juni** reist du gern **nach Finnland**.

> **Rule:** Adverbials of **time** come before **manner/object**, which come before **place**.
