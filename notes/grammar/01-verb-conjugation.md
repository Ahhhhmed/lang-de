# Verb Conjugation

## Stem Changes & Reflexive Verbs

```
e → i / ie                  Reflexiv                    sein
──────────                  ────────                    ────
a → ä

ich          -e    helfe    ich      wasche mich         bin
┌────────────────────────────────┐
│ du         -st   hilfst   ä   │    du       wäschst dich        bist
│ er                             │    er
│ sie        -t    hilft    ä   │    sie      wäscht sich         ist
│ es                             │    es
└────────────────────────────────┘
wir          -en   helfen       wir      waschen uns         sind
ihr          -t    helft        ihr      wascht euch         seid
sie/Sie      -en   helfen       sie/Sie  waschen sich        sind
```

> **Note:** The `e → i/ie` and `a → ä` stem changes only occur in the **du/er/sie/es** forms.

## Irregular Verbs: nehmen & treffen (Präsens)

```
nehmen          treffen
ich nehme       treffe
du nimmst       triffst
er nimmt        trifft
wir nehmen      treffen
ihr nehmt       trefft
sie nehmen      treffen
```

## Reflexive Pronouns

```
           Reflexiv (Akk.)    Dativ Reflexiv
       ┌──────────────────────────────────┐
ich    │   mich                mir        │
du     │   dich                dir        │
       └──────────────────────────────────┘
er/sie/es  sich                sich
wir        uns                 uns
ihr        euch                euch
sie/Sie    sich                sich
```

> Use **Dativ** reflexive when there is already a direct object (Akkusativ):  
> `Ich wasche mir die Hände.` *(I wash my hands)*  
> `Ich wasche mich.` *(I wash myself)*
