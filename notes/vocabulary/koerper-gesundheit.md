### Sleep, Care Products & Hygiene
- **die Ruhe / der Schlaf / der Traum** = rest / sleep / dream
- **Gute Besserung!** = Get well soon!
- **Gesundheit!** = Bless you / Health!
- **das Vitamin** = vitamin
- **die Seife / das Shampoo** = soap / shampoo
- **das Handtuch / das Badetuch** = towel / bath towel
- **die (Nagel-)Schere / die Bürste / der Kamm** = nail scissors / brush / comb
- **die Zahnbürste / die Zahnpasta** = toothbrush / toothpaste
- **der Rasierapparat / der Spiegel / der Lippenstift** = razor / mirror / lipstick
- **der Föhn / die Haare föhnen** = hair dryer / to blow-dry hair
- **das Taschentuch** = tissue / handkerchief
- **sauber / rein / gepflegt** = clean / pure / well-groomed
- **schmutzig / dreckig / ungepflegt** = dirty / filthy / neglected
- **gesund / fit / die Gesundheit / die Fitness** = healthy / fit / health / fitness
- **krank / die Krankheit** = sick / illness
- **der Atem / atmen** = breath / to breathe
- **das Schmerzmittel** = painkiller
- **schwer/leicht verletzt** = seriously/lightly injured
- **die (Drogen-)Beratung** = drug counseling
- **sich versichern lassen** = to get insured
